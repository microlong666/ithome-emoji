### 表情包使用

#### 1、保存为图片直接使用

#### 2、使用 jsDelivr CDN 服务

格式：

`https://cdn.jsdelivr.net/gh/microlong666/ithome-emoji/1(数字1到141).svg`

示例：

`https://cdn.jsdelivr.net/gh/microlong666/ithome-emoji/1.svg`

`https://cdn.jsdelivr.net/gh/microlong666/ithome-emoji/25.svg`
